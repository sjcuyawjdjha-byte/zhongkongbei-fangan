\begin{reportframe}
    \sectiontitle{1、机器人机械设计方案}

    \mybody
    分为上、下车机械结构分别阐述。启动时上车置于下车之上。

    \begin{center}
        \vspace{0.5em}
        \begin{minipage}{0.48\textwidth}
            \centering
            \includegraphics[height=0.22\textheight]{image3.png}
            \vspace{0.5em}
            \captionof{figure}{下车底板结构示意图}
        \end{minipage}
        \hfill
        \begin{minipage}{0.48\textwidth}
            \centering
            \includegraphics[height=0.22\textheight]{image2.png}
            \vspace{0.5em}
            \captionof{figure}{下车顶板结构示意图}
        \end{minipage}
        \vspace{0.5em}
    \end{center}

    \subsectiontitle{（1）底板设计（见图 1）}
    底板作为下车的动力基准，采用高强度板材激光切割而成。其表面预留了高密度的安装孔位阵列，确保驱动电机座与万向轮支撑件的装配精度。通过将蓄电池等重型组件低重心布置，使整车重心高度处于安全包络内，显著提升了机器人高速转向时的动态稳定性。

    \subsectiontitle{（2）顶板设计（见图 2）}
    顶板兼具结构加强与子车承载功能。
    \begin{itemize}[leftmargin=2em, itemsep=3pt, parsep=0pt, topsep=4pt]
        \item \textbf{功能腰型槽}：中心布置的三组腰型槽在实现拓扑轻量化的同时，为子车的锁紧机构提供了 $\pm 25$\,mm 的机械调节冗余，增强了系统适配性。
        \item \textbf{90$^\circ$ 折弯工艺}：侧边采用折弯补强结构，极大地提高了板材的抗弯惯性矩，有效防止子车在载入对接过程中因局部受压而产生形变。
    \end{itemize}

    \subsectiontitle{（3）底盘工艺指标}
    板材选用 6061-T6 铝合金，加工公差控制在 $\pm 0.1$\,mm 级别。所有边缘均经过倒角处理，并预留了电气走线孔，实现了机械结构与电路系统的模块化集成。

    \subsectiontitle{（4）专用物块推动机构设计（见图 3）}
    推动机构（推块臂）是子母车方案中用于精确、一致地推动物块的核心执行部件。为避免因机构误差导致多车协同推动时物块倾斜或滑落，整体采用了非线性几何与严苛的公差设计。

    \begin{center}
        \vspace{0.5em}
        \includegraphics[height=0.22\textheight]{image4.png}
        \vspace{0.5em}
        \captionof{figure}{物块推动机构三维结构示意图}
        \vspace{0.3em}
    \end{center}

    \begin{itemize}[leftmargin=2em, itemsep=3pt, parsep=0pt, topsep=4pt]
        \item \textbf{多节弯曲避障臂}：臂体采用特定的非线性弯曲几何形状，在有效避让物块上方潜在障碍物的同时，优化了刚性分布。经有限元分析（FEA）验证，在受侧向推力时其弹性形变 $\delta \leq 0.05$\,mm。
        \item \textbf{高精度法兰与执行推块}：顶部法兰通过四个沉头孔与车体刚性连接，保证极大的扭转惯性矩。底部执行推块（基准面 1）的宽度公差严格控制在 $\pm 0.50$\,mm 以内，接触面垂直度公差 $\leq \pm 0.1^\circ$，确保动作的正交性与一致性。
        \item \textbf{材料与表面处理}：主体采用 6061-T6 铝合金进行五轴 CNC 精密加工，以降低运动惯量。接触面经阳极氧化处理并添加 PTFE（聚四氟乙烯）涂层，显著降低磨损，保障长期作业精度。
    \end{itemize}

\begin{reportframe}
    \sectiontitle{3、机器人软件设计方案}

    \mybody
    软件系统基于 STM32 平台，采用 C 语言开发，以模块化方式组织代码，将底盘控制、巡线、视觉通信与机械臂控制分别封装为独立功能模块，由主程序统一调度。

    \subsectiontitle{（1）整体架构}
    主程序通过状态机管理机器人行为，定义若干运行状态（如 \texttt{LINE\_FOLLOWING}、\texttt{VISION\_TRACKING}、\texttt{ARM\_GRABBING} 等），每个控制周期根据当前状态调用对应的功能函数，实现任务的有序切换与执行。

    \subsectiontitle{（2）底盘运动控制}
    基于麦克纳姆轮运动学模型，推导四轮转速与整车速度向量（$v_x$、$v_y$、$\omega$）之间的映射关系，封装为 \texttt{Chassis\_SetVelocity()} 函数。结合 MG513 编码器的转速反馈，对各轮分别实施 PID 闭环调速，保证直行与全向移动的精度。

    \subsectiontitle{（3）巡线模块}
    读取多路红外传感器的数字电平，通过加权偏差算法计算当前偏移量，将偏差输入 PID 控制器，输出左右侧电机的差速修正量，实现稳定巡线。当传感器检测到十字路口特征时，触发路口判断逻辑，执行预设的直行、转弯或停车动作。

    \subsectiontitle{（4）视觉融合}
    K30 视觉模块通过串口中断方式向 STM32 发送目标坐标帧。主控解析坐标后，在视觉追踪状态下计算目标与图像中心的像素偏差，换算为底盘横移与旋转指令，驱动小车对准目标，随后切换至机械臂抓取状态。

    \subsectiontitle{（5）机械臂控制}
    升降步进电机通过脉冲数控制丝杆位移，封装为 \texttt{Arm\_MoveTo(height)} 函数；舵机夹取通过 \texttt{Servo\_SetAngle()} 设定张合角度。抓取流程为：下降至目标高度 $\rightarrow$ 舵机夹紧 $\rightarrow$ 上升至安全高度 $\rightarrow$ 运送至目标区域 $\rightarrow$ 舵机释放。

    \subsectiontitle{（6）已完成功能}
    目前已完成巡线、路口判断、底盘全向运动、视觉串口解析等基础功能，机械臂抓取流程与完整状态机仍在调试优化中，以最终参赛代码为准。

    \vspace*{\fill}
\end{reportframe}

\begin{reportframe}
    \sectiontitle{4、机器人完赛流程及基本功能实现方案}

    \mybody
    本机器人为单车设计，启动后按照预设策略自主完成比赛全程，整体流程分为以下阶段：

    \subsectiontitle{（1）启动阶段}
    裁判发出开始信号后，机器人完成 STM32 系统自检与各模块初始化，确认巡线传感器、K30 视觉模块及机械臂均处于正常工作状态，随即进入巡线行进模式，驶离启动区。

    \subsectiontitle{（2）目标搜寻阶段}
    机器人沿地面引导线巡线行驶，K30 视觉模块持续扫描周围环境。一旦识别到目标物体（方块或圆环），主控切换至视觉追踪模式，驱动底盘横移对准目标，精确定位后触发抓取流程。

    \subsectiontitle{（3）抓取阶段}
    底盘对准目标后，机械臂按照预设动作序列执行抓取：丝杆下降至对应高度，舵机夹紧目标物体，随后丝杆上升至安全高度，完成稳定夹持。

    \subsectiontitle{（4）运送与放置阶段}
    机器人夹持目标物体后重新切换至巡线模式，按路径规划函数输出的路线行驶至目标得分区（二分区或三分区）。到达目标位置后，机械臂下降，舵机松开，将物体放置于得分区内，完成得分动作。

    \subsectiontitle{（5）循环执行}
    完成一次放置后，机器人重新进入目标搜寻阶段，按照上述流程循环往复，直至比赛时间结束。

    \subsectiontitle{（6）核心功能实现要点}
    \begin{itemize}[leftmargin=2em, itemsep=3pt, parsep=0pt, topsep=4pt]
        \item \textbf{巡线稳定性}：采用 PID 差速控制，确保在不同路面条件下均能可靠追线，避免偏轨。
        \item \textbf{视觉定位精度}：K30 模块输出目标像素坐标，主控通过比例控制将底盘对准目标，减小抓取误差。
        \item \textbf{机械臂重复定位}：步进电机配合限位检测，保证每次抓取高度一致，提升成功率。
        \item \textbf{状态机鲁棒性}：各功能模块间通过状态机严格切换，避免多任务竞争导致的指令冲突。
    \end{itemize}

    \vspace{0.8em}
    综上，本方案通过巡线与视觉的有机融合，结合 STM32 的实时调度能力，实现了一套具备自主搜索、精准抓取与高效运送能力的完整竞赛解决方案。

    \vspace*{\fill}
\end{reportframe}
